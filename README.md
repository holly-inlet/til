### 2022/05

[TIL —— 2022/05](2022/2022-05-til.md)

- 就活の進展が見られない上に，卒論のスケジュールも厳しくなってきたので，これで最後と考え4社の選考に挑んだ

  - 会社説明会　1社

  - 読む論文をピックアップ
  - ピックアップした論文の引用文献をリスト化
  - 所属する研究室のミーティングで進捗を発表
- このTILの用途・目的を変更
